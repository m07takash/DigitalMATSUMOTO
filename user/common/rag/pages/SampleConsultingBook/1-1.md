# 外部環境分析（PEST・5Forces）

## 概要

外部環境分析は、自社ではコントロールできないマクロ環境と業界構造を把握するための分析手法である。PEST分析でマクロ環境を、5Forces分析で業界の競争環境を体系的に分析する。

## PEST分析の4要素

### P（Political：政治的要因）
- 法規制の変更、税制改正、貿易政策、政治的安定性
- **実施のコツ：** 直近3年の法改正と今後の審議中法案を網羅的にリストアップする

### E（Economic：経済的要因）
- GDP成長率、為替変動、金利動向、失業率、インフレ率
- **実施のコツ：** 自社事業に直結する経済指標を3〜5つに絞り、時系列でトレンドを把握する

### S（Social：社会的要因）
- 人口動態、ライフスタイル変化、価値観の変容、教育水準
- **実施のコツ：** 定量データだけでなく、消費者調査やSNSトレンドなど定性情報も活用する

### T（Technological：技術的要因）
- 新技術の登場、技術の標準化、R&D投資動向、特許出願動向
- **実施のコツ：** 技術の成熟度をガートナーのハイプサイクルなどで評価し、実用化時期を見極める

## 5Forces分析の5つの力

| 力 | 分析ポイント |
|---|---|
| 業界内の競争 | 競合数、成長率、差別化の程度、固定費比率 |
| 新規参入の脅威 | 参入障壁の高さ、初期投資額、規制の有無 |
| 代替品の脅威 | スイッチングコスト、代替品の価格性能比 |
| 買い手の交渉力 | 顧客の集中度、情報の非対称性、後方統合リスク |
| 売り手の交渉力 | サプライヤーの集中度、原材料の希少性、前方統合リスク |

## 実施手順

1. **情報収集（1週間）：** 公開情報、業界レポート、専門家インタビューからデータを収集
2. **要因の整理（2〜3日）：** 各フレームワークに沿って要因を分類・整理
3. **影響度評価（2〜3日）：** 各要因が自社に与える影響度を「高・中・低」で評価
4. **示唆の導出（1〜2日）：** 「機会」と「脅威」を明確に言語化し、戦略策定への入力とする

## 注意点

- PEST分析と5Forces分析は補完関係にある。マクロ要因が業界構造にどう影響するかを紐付けて考えることで、より深い洞察が得られる。
- 分析が目的化しないよう、常に「自社にとっての意味」を問い続けることが重要である。
