# 戦略立案の全体像

## はじめに

戦略立案は、企業が持続的な競争優位を確立するための根幹となる活動である。本章では、戦略立案を「環境分析」「戦略策定」「実行計画」の3フェーズに分解し、各フェーズの目的・手法・アウトプットを体系的に整理する。

## 3フェーズフレームワーク

### フェーズ1：環境分析（2〜4週間）

**目的：** 自社を取り巻く外部環境と内部環境の現状を客観的に把握する。

**主な手法：**
- 外部環境：PEST分析、5Forces分析、市場動向調査
- 内部環境：VRIO分析、バリューチェーン分析、財務分析

**アウトプット：** 環境分析レポート（機会・脅威・強み・弱みの一覧）

### フェーズ2：戦略策定（3〜6週間）

**目的：** 環境分析の結果を踏まえ、複数の戦略オプションを導出し、最適な戦略を選択する。

**主な手法：**
- クロスSWOT分析による戦略オプション導出
- 実現可能性・収益性・リスクの3軸評価
- 経営層とのワークショップによる合意形成

**アウトプット：** 中期経営戦略書（3〜5年のロードマップ付き）

### フェーズ3：実行計画（2〜4週間）

**目的：** 選択した戦略を具体的なアクションプランに落とし込み、実行体制を構築する。

**主な手法：**
- 施策のWBS（Work Breakdown Structure）分解
- KPI設計とマイルストーン設定
- リソース配分とスケジュール策定

**アウトプット：** 実行計画書（担当・期限・KPI・予算を明記）

## 実務上のポイント

1. **経営層の早期巻き込み：** フェーズ1の段階から経営層をスポンサーとして設定し、定期報告を行うことで、最終提言の受容性を高める。
2. **仮説駆動アプローチ：** 分析に入る前に「初期仮説」を立て、検証すべき論点を明確にすることで、分析の効率と精度を両立させる。
3. **イテレーション：** 3フェーズは一方向ではなく、必要に応じて前のフェーズに戻るイテレーティブなプロセスとして運用する。

## まとめ

戦略立案の成功には、フレームワークの正しい理解と柔軟な適用が不可欠である。形式的な分析に陥らず、常に「So What?（だから何なのか）」を問い続ける姿勢が、質の高い戦略を生み出す鍵となる。
