# 内部環境分析（VRIO・バリューチェーン）

## 概要

内部環境分析は、自社が保有する経営資源と活動プロセスを評価し、競争優位の源泉を特定するための分析手法である。VRIO分析で経営資源の競争優位性を、バリューチェーン分析で価値創造プロセスの強み・弱みを明らかにする。

## VRIO分析の4条件

VRIO分析は、以下の4つの問いに順番に答えることで、経営資源の競争優位性を評価する。

### V（Value：経済的価値）
- その資源は、外部環境の機会を活用したり脅威を無力化したりできるか？
- 「No」の場合：競争劣位の状態

### R（Rarity：希少性）
- その資源を保有している競合は少数か？
- 「No」の場合：競争均衡の状態（持っていて当然の資源）

### I（Imitability：模倣困難性）
- その資源を持たない企業が、獲得・開発するのにコスト上の不利が生じるか？
- 模倣困難性の源泉：歴史的経路依存性、因果曖昧性、社会的複雑性、特許

### O（Organization：組織体制）
- その資源を最大限に活用するための組織体制（方針・手続き・仕組み）が整っているか？
- 4条件すべてを満たす場合：持続的競争優位

## バリューチェーン分析

### 主活動（5つ）
1. **購買物流：** 原材料の調達・保管・在庫管理
2. **製造：** 原材料を最終製品に変換するプロセス
3. **出荷物流：** 完成品の保管・配送・流通
4. **マーケティング・販売：** 顧客への訴求と販売活動
5. **サービス：** 製品価値を維持・向上させるアフターサービス

### 支援活動（4つ）
1. **全般管理：** 経営企画、財務、法務、経理
2. **人的資源管理：** 採用、育成、評価、報酬制度
3. **技術開発：** R&D、製品設計、プロセス改善
4. **調達活動：** サプライヤー選定、価格交渉、品質管理

## 実施のコツ

- **定量化を意識する：** 各活動のコスト構造を可能な限り数値で把握し、競合や業界平均と比較する
- **顧客視点を忘れない：** バリューチェーン上のどの活動が顧客にとっての価値を最も生んでいるかを明確にする
- **VRIOとの連携：** バリューチェーンで特定した強みをVRIOの4条件で評価し、持続可能性を判断する

## まとめ

内部環境分析は「自社の真の強み」を客観的に特定するプロセスである。思い込みを排し、データと論理に基づいた評価を行うことが、有効な戦略策定の前提条件となる。
