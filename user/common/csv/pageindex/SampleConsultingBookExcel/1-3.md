# 3C分析

事業の成功要因（KSF: Key Success Factor）を、3つの観点から構造的に把握するフレームワーク（大前研一）。

## 3つのC

| C | 着眼点 | 主な問い |
|---|--------|----------|
| **Customer**（市場・顧客） | 市場規模・成長率・ニーズ・購買行動 | 誰が、何を、なぜ買うのか？ |
| **Competitor**（競合） | 競合各社のシェア・戦略・強み・参入動向 | どこが勝っていて、なぜか？ |
| **Company**（自社） | 自社の経営資源・ケイパビリティ・財務 | 何で勝てて、何が足りないか？ |

## 進め方

1. **Customer起点で始める** — 自社視点から始めると「あれば良い差別化」に流れやすい
2. **Competitorで相対評価** — 強み・弱みは「絶対値」ではなく「競合比」で意味を持つ
3. **Companyで打ち手化** — 顧客ニーズ × 競合との差を、自社資源でどう埋めるかに落とす

## 4Cへの拡張

外部環境（Channel／協業先など）を加えた4C分析もある。プラットフォーム型・流通介在型ビジネスで有効。

## よくある失敗

- 3つのCを「並列に並べて満足」してしまう。本来は **Customer→Competitor→Company** の因果順で並べる
- データの粒度がC間でバラバラ（市場全体 vs 一部地域）
- 「自社の強み」を商品スペックで語り、顧客価値に翻訳できていない
