# バリューチェーン分析

事業活動を「価値を生む一連のつながり」として分解し、競争優位の源泉と改善余地を特定する手法（Porter）。

## 主活動と支援活動

**主活動**（直接的な価値創出）
1. 購買物流（Inbound Logistics）
2. 製造（Operations）
3. 出荷物流（Outbound Logistics）
4. マーケティング・販売
5. サービス

**支援活動**（主活動を支える）
- 全般管理（経営インフラ）
- 人事・労務管理
- 技術開発（R&D・IT）
- 調達

## マージンの考え方

各活動が生む価値の合計から総コストを引いたものが「マージン（利益）」。マージンを最大化するには、

- **コスト優位戦略**：各活動のコストを徹底的に削減
- **差別化戦略**：特定活動で他社と違う価値を生む

のどちらか（あるいは両方）を選ぶ。

## 分析の進め方

1. 自社のバリューチェーンを描く（業界標準ではなく **自社実態** で）
2. 各活動の **コスト構造** を整理
3. 各活動の **顧客価値への貢献度** を評価
4. 「コストは高いが価値は低い」「価値が高いのに投資不足」の活動を特定
5. **コアコンピタンス**（差別化の源泉）を強化、非コア活動は外注検討

## バリューチェーンとサプライチェーンの違い

- **バリューチェーン**：自社内の価値創出の流れ（戦略視点）
- **サプライチェーン**：原材料から消費者までのモノの流れ（オペレーション視点）

## よくある失敗

- 業界一般のテンプレートを当てはめ、自社特有の活動構造を見落とす
- 各活動を独立して見て、活動間の **連携・相乗効果** を見逃す
- コスト削減ばかりに目が行き、差別化機会を逃す
