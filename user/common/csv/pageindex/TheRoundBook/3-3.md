3-3 仮説づくりと検証（著者：Domon）

問いが定まったら、次は仮説です。問いに対して、「たぶんこういうことだろう」「おそらくこの構造で動いているはずだ」という仮の答えを置く。この作業が、検討を前に進める推進力になります。

仮説がないまま情報を集めると、たいてい迷子になります。何を見ても関係がありそうに見えるし、何を見ても決め手に欠ける。集めた情報が積み上がる割に、判断には近づかない。これは現場でよく見る光景です。逆に、粗くてもいいから仮説が置いてあると、見るべき情報の優先順位が決まります。仮説に対して効く情報と、効かない情報が分かれてくる。検討の解像度は、情報量ではなく、仮説の有無で決まると言ってもいいくらいです。

ここで一番大事なことを先に言っておきます。仮説の質を決めるのは、当たっているかどうかではありません。反証可能であるかどうかです。

これは少し馴染みのない言い方かもしれません。普通、仮説と言えば「当たっていそうなもの」を置きたくなります。けれど、当たっていそうな仮説の中には、何が起きても「当たっている」と解釈できてしまう、つまり否定しようのない仮説が混ざっていることがあります。「市場環境が厳しいから業績が悪い」みたいな仮説は、一見もっともらしいのですが、業績が良くなれば「市場環境の中でも善戦した」と説明できるし、悪くなれば「やはり市場環境が厳しかった」と説明できる。どんな結果でも仮説を維持できてしまうものは、検討の足場として機能しません。

反証可能であるとは、「もしこの情報が出てきたら、この仮説は間違いだったと判断する」という条件をあらかじめ言える、ということです。仮説と一緒に、「これが見えたら撤回する」という線を引いておく。この線が引けない仮説は、どれだけ精緻に組み立てても、検証の対象になりません。検証されない仮説は、信念や願望と区別がつかなくなっていきます。

仮説を置く時に意識しているのは、いくつかあります。まず、仮説は一つに絞らないこと。最初から一つの仮説に賭けると、その仮説に都合のいい情報ばかりが目に入るようになります。これは意志の問題ではなく、人間の認知が自然にそうなる、という構造の話です。だから、複数の仮説を並べて置く。「Aかもしれないし、Bかもしれない。Cの可能性もある」と、最初は広めに張っておく。検討が進む中で、どの仮説が残るかが見えてきます。

次に、仮説には根拠と、その根拠が崩れる条件をセットにすること。「Aだと考える根拠はこれこれ。ただし、こういう事実が出てくればAは怪しくなる」というところまで一緒に書く。これをやっておくと、検討中に新しい情報が出てきた時、仮説をどう動かすかの判断が早くなります。情報に振り回されるのではなく、情報を仮説の更新に使えるようになります。

それから、仮説は粗くていい、ということ。最初から精緻な仮説を作ろうとすると、たいてい手が止まります。粗くていいから、まず置く。置いてから磨く。これは順番の問題で、磨いてから置こうとすると、磨くための材料が足りない、という袋小路に入ります。とりあえずの仮説を置いて、検証しながら更新していく方が、結果的には速いし、深いところに届きます。

仮説の置き方には、その人の経験や思考の癖が出ます。経験豊富な人ほど、最初の仮説が当たりやすい。これは事実だと思います。ただ、経験は同時に思い込みの源泉にもなります。過去の成功体験で当たった仮説が、目の前の状況でも当たるとは限らない。経験を活かしながら、経験に引きずられない。このバランスを保つために、複数仮説と反証条件のセットが効きます。一つの仮説に賭けないこと、撤回条件をあらかじめ言葉にしておくこと。この二つが、経験の落とし穴から自分を守ってくれます。

検証の透明性についても、ここで触れておきます。仮説を検証する時、その過程を周囲に見せられる形にしておくことが大事です。「自分はAだと思った、なぜなら根拠はこれで、これが崩れたらBに動く」というプロセスを、頭の中だけで完結させない。紙でもホワイトボードでもチャットでもいい、見える場所に置く。これをやっておくと、議論が「結論の正しさ」ではなく「考え方の筋」を巡るものになります。結論で殴り合うと議論は壊れますが、考え方の筋を共有できると、議論は積み上がります。Professionalが意思決定者の隣に立つ時、自分の思考を透明にしておくことは、信頼の通貨のようなものだと思っています。

仮説をつくる作業は、慣れないうちは少し怖いものです。間違うかもしれないものを口に出すのは、勇気がいる。でも、間違うかもしれないものを置かない限り、議論は前に進みません。仮説を置くというのは、「これは違うかもしれない」と引き受けることでもあります。引き受けたものだけが、検証され、磨かれ、判断に変わっていく。仮説を置く勇気は、Professionalの基礎体力の一つだと感じています。
