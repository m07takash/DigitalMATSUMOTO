# 設定ファイルとデプロイ構成

## 概要

システムの動作はYAML/ENV/JSONの3種類の設定ファイルで制御される。Dockerベースのデプロイで、`startup.sh`からStreamlit UIとFastAPIを同時起動する構成が標準。

## 仕組み

### 設定ファイル

| ファイル | 形式 | 役割 |
|---------|------|------|
| `setting.yaml` | YAML | フォルダパス、Web検索モデル、URL_FETCH等の構造設定 |
| `system.env` | ENV | APIキー、認証設定、マスターファイル名（`system.env_sample` をコピーして作成） |
| `startup.sh` | Shell | 起動スクリプト（`startup.sh_sample` をコピーして作成、`.gitignore` 済み） |
| `user/common/agent/*.json` | JSON | エージェント定義（人格・LLM・ナレッジ） |
| `user/common/mst/*.json` | JSON | マスターデータ（ユーザー・RAG・プロンプト・カテゴリ・Notion・scheduled_jobs） |

### 主要環境変数

| 変数名 | 説明 |
|--------|------|
| `OPENAI_API_KEY` | OpenAI APIキー（GPT + Embedding） |
| `GEMINI_API_KEY` / `ANTHROPIC_API_KEY` / `XAI_API_KEY` / `LLAMA_API_KEY` | 各LLMのAPIキー |
| `AZURE_OPENAI_ENDPOINT` / `AZURE_OPENAI_API_KEY` / `AZURE_OPENAI_API_VERSION` / `AZURE_OPENAI_EMBED_MODEL` | Azure OpenAI Service（chat/image/embedding） |
| `LOGIN_ENABLE_FLG` / `LOGIN_AUTH_METHOD` | ログイン認証の有効化と方式（`JSON` / `RDB`） |
| `EMBEDDING_MODEL` | 埋め込みモデル（既定 `text-embedding-3-large`） |
| `USER_MST_FILE` / `RAG_MST_FILE` / `PROMPT_TEMPLATE_MST_FILE` / `NOTION_MST_FILE` | マスターファイル名 |
| `AGENT_PERSONA_SOURCE` | ペルソナソース（`EXCEL` / `RDB` / `BOTH`） |
| `USER_MEMORY_*_BACKEND` | ユーザーメモリ各層の保存先（`EXCEL` / `NOTION` / `RDB`） |
| `POSTGRES_*` | PostgreSQL接続情報（分析DB / RDB認証 / ベクトル化） |

### Docker構成

`Dockerfile`はUbuntu 22.04ベースで、Python3、MeCab + NEologd辞書、日本語フォント（IPAex/Noto CJK）を含む。ポート8501（Streamlit）/ 8899（FastAPI）を公開する。デフォルトCMDは `bash startup.sh`。

`startup.sh_sample` は FastAPI（gunicorn + uvicorn workers、8899）をバックグラウンド起動 → Streamlit（WebDigiMatsuAgent.py、8501）をフォアグラウンドで起動する構成。Nginxリバースプロキシ + Let's Encrypt でのHTTPS運用時はWebSocket（`Upgrade`/`Connection`）と長めの`proxy_read_timeout`が必須。

## 設定場所

- `setting.yaml` / `system.env` / `startup.sh` / `Dockerfile`: プロジェクトルート

## 設定例

```bash
# 雛形をコピー
cp system.env_sample system.env
cp startup.sh_sample startup.sh

# コンテナ起動（待機状態）
docker run -dit --name digimatsumoto \
  -p 8501:8501 -p 8899:8899 \
  -v $(pwd):/app/DigitalMATSUMOTO \
  -w /app/DigitalMATSUMOTO \
  digimatsumoto bash

# サービス一括起動
docker exec -it digimatsumoto bash startup.sh
```
