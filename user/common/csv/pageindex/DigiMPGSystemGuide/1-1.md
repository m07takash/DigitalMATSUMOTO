# マルチLLMサポート

## 概要

`DigiM_FoundationModel.py`がLLM抽象化レイヤーとして機能し、GPT / Gemini / Claude / Grok / Llama / Azure OpenAI Service の各系統を統一インターフェースで呼び出す。エージェントJSONのENGINE設定でモデルを切り替える。

## 仕組み

### ディスパッチ機構

`call_function_by_name(func_name, *args, **kwargs)` が関数名文字列からPython関数を動的に呼び出す。エージェントJSONの `FUNC_NAME` フィールドが関数名を指定する。

### 各LLM関数（テキスト生成）

| 関数名 | 対応LLM | ストリーミング |
|--------|---------|-------------|
| `generate_response_T_gpt` | OpenAI GPT | 対応 |
| `generate_response_T_azure_openai` | Azure OpenAI Service（gpt-*） | 対応 |
| `generate_response_T_gpt_response` | OpenAI Responses API | 非対応 |
| `generate_response_openai_tool` | OpenAI Tools | 対応 |
| `generate_response_T_gemini` | Google Gemini | 対応（失敗時は非ストリーム再実行） |
| `generate_response_T_claude` | Anthropic Claude | 対応 |
| `generate_response_T_grok` | xAI Grok | 対応 |
| `generate_response_T_llama` | Llama API（llamaapi） | 非対応 |

全関数は共通シグネチャ `(query, system_prompt, model, memories, image_paths, agent_tools, stream_mode)` を持ち、`yield prompt, response, completion` でジェネレータとして結果を返す。

### クライアント管理

各LLMクライアントは `_clients` 辞書でシングルトン管理され、初回呼び出し時にのみ生成される。Azure OpenAIは `_get_azure_openai_client(api_version=...)` でAPIバージョンごとに別キーで管理し、エージェントJSONの `PARAMETER.api_version` でエンジン単位の上書きが可能。

## 設定場所

- エージェントJSON: `user/common/agent/*.json` 内の `ENGINE` セクション
- APIキー: `system.env` の `OPENAI_API_KEY` / `GEMINI_API_KEY` / `ANTHROPIC_API_KEY` / `XAI_API_KEY` / `LLAMA_API_KEY`
- Azure共通: `AZURE_OPENAI_ENDPOINT` / `AZURE_OPENAI_API_KEY` / `AZURE_OPENAI_API_VERSION`

## 設定例

```json
{
  "ENGINE": {
    "LLM": {
      "DEFAULT": "GPT-4.1-mini",
      "GPT-4.1-mini": {
        "NAME": "GPT-4.1-mini",
        "FUNC_NAME": "generate_response_T_gpt",
        "MODEL": "gpt-4.1-mini",
        "PARAMETER": {},
        "TOKENIZER": "tiktoken",
        "MEMORY": {"limit": 10000, "role": "both", "priority": "latest", "similarity_logic": "cosine", "digest": "Y"}
      },
      "GPT-Azure": {
        "NAME": "GPT-Azure",
        "FUNC_NAME": "generate_response_T_azure_openai",
        "MODEL": "my-gpt5-deployment",
        "PARAMETER": {"api_version": "2025-04-01-preview"},
        "TOKENIZER": "tiktoken",
        "MEMORY": {"limit": 10000, "role": "both"}
      }
    }
  }
}
```
