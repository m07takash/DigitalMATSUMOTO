# 会話履歴・フィードバック保存

## 概要
セッション内の会話履歴をJSONファイルとして保存・管理し、フィードバックデータをCSVまたはNotionに書き出す機能。階層的ユーザーメモリ（History/Nowaday/Persona）の生成入力にもなる（詳細は3-10）。

## 仕組み

### 会話履歴の保存（`save_history_batch`）
`DigiM_Session.py`の`save_history_batch()`がメインの保存関数。スレッドセーフなファイルロック（`_get_file_lock`）を使用して`chat_memory.json`を読み書きする。

- **seq/sub_seq構造**: 会話は`seq`（メイントピック番号）と`sub_seq`（サブステップ番号）の2階層で管理される。各seqには`SETTING`ブロックが付随する。
- **SETTINGの主要フラグ**:
  - `FLG`（`"Y"`/`"N"`）: 論理削除フラグ。`"N"`のseqは画面表示にもメモリ参照にも使われない（`get_history_active`で除外）。`chg_seq_history()`で更新。
  - `MEMORY_FLG`（`"Y"`/`"N"`、未設定時は`"Y"`扱い）: メモリ参照フラグ。`"N"`のseqは**画面には残るがLLMコンテキスト（メモリ）には含まれない**。`chg_seq_memory_flg()`で更新。`get_memory()`内で除外される。
- **保存内容**: `sub_seq_data`に`setting`/`prompt`/`response`/`digest`/`log`/`feedback`等のデータを格納。
- **DB Export連携**: 新規保存時、DB ExportステータスがDONEならUNDOにリセットする。`digim_dialogs.flg` / `digim_dialogs.memory_flg` 列にエクスポート時に反映される。

### フィードバック保存（`DigiM_GeneFeedback.py`）
`create_feedback_data()`がフィードバックデータを収集・保存する。

- 会話履歴内の`feedback`ブロックからフラグが立っている項目を抽出。
- `FEEDBACK.SAVE_MODE`で保存先を選択（`"CSV"` or `"Notion"`）。
- CSV保存: `save_feedback_csv()`でtitle列をキーに既存行を上書きまたは追加。
- Notion保存: `save_feedback_notion()`でページ作成後、FIELD_MAPに基づき型別にプロパティを更新。
- `FIELD_MAP`で内部キー/列名/Notionプロパティ名/型のマッピングを定義。

### ユーザーメモリHistory生成（`DigiM_GeneUserMemory.py`）
セッションの会話履歴+Feedbackから、ユーザーを理解するための要点をLLMで構造化抽出（topic/excerpt/axis_tags/confidence）し、Excel/Notion/RDBのHistory DBへ保存する。後段のNowaday→Personaマージへつながる。詳細は3-10。

`DigiM_GeneUserDialog.py` は旧UserDialog互換シムとして残るのみ（`DigiM_GeneUserMemory.save_history_for_unsaved_sessions` を呼ぶだけのスタブ）。

## 設定場所
- 会話履歴ファイル: `setting.yaml`の`SESSION_FILE_NAME`（デフォルト: `chat_memory.json`）
- フィードバック設定: エージェントJSON の `FEEDBACK`（SAVE_MODE, SAVE_DB, FIELD_MAP）
- ユーザーメモリ: 環境変数 `USER_MEMORY_HISTORY_BACKEND` / `USER_MEMORY_NOWADAY_BACKEND` / `USER_MEMORY_PERSONA_BACKEND`(EXCEL/NOTION/RDB)、`USER_MEMORY_DEFAULT_LAYERS`、`USER_MEMORY_AUTO_APPROVE_THRESHOLD` 他（3-10参照）
- CSVフォルダ: `setting.yaml`の`RAG_DATA_CSV_FOLDER`

## 設定例
```json
"FEEDBACK": {
  "ACTIVE": "Y",
  "SAVE_MODE": "CSV",
  "SAVE_DB": "Sample00_Feedback",
  "DEFAULT_CATEGORY": "未設定",
  "FEEDBACK_ITEM_LIST": ["memo", "comment"]
}
```
