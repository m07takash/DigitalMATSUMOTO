# WebUI チャット画面

## 概要

Streamlitベースのチャット画面（WebDigiMatsuAgent.py）。サイドバーでエージェント・エンジン選択やセッション管理を行い、メインエリアでチャット・ファイル添付・各種モード切替を提供する。

## 仕組み

**サイドバー構成:**

- **Select Agent:** エージェントJSONをselectboxで選択。選択時にagent_data・engine_nameを自動更新
- **New Chat / Refresh List:** セッション新規発番・一覧更新ボタン
- **Engines expander:** LLMエンジンとIMAGEGENエンジンをselectboxで切替
- **Sessions expander:** セッション管理（詳細は5-3参照）
- **Analytics expander:** Knowledge Utility分析モード・次元削減手法の設定、Insight/Monthly分析実行
- **RAG Management expander:** RAGデータ更新・削除、Page Index Export、User Memory管理者ブロック（Target User IDs multiselect / Period date / 統合 Update User Memory ボタン: History→Nowaday→Persona）
- **Web API expander:** FastAPIサーバーの起動/停止/ヘルスチェック

**メインエリア構成:**

- **Chat Name:** セッション名の表示・変更
- **Time Setting:** Real Date / Custom Date / No Dateの3モード切替
- **Exec Setting:** Streaming Mode, Memory Use, Save Digest, Magic Word, Memory Similarityのチェックボックス群
- **RAG Setting:** RAG Query Gen, Meta Searchのチェックボックス
- **History Seq Visible:** LATEST(件数指定)/FULL切替、ALL/SUMMARY表示モード
- **Delete Chat History (Chk):** 各seqの「Delete(seq:K)」チェックを集めて `chg_seq_history(seq, "N")` を実行。選択seqは画面とメモリ参照の両方から除外される
- **Memory Off(seq:K):** 各seq詳細パネル内のチェックボックス（即時反映）。ONで `chg_seq_memory_flg(seq, "N")` を呼び、画面表示は残したまま LLM のメモリ参照からだけ除外する
- **チャット表示:** seq/sub_seq単位で会話履歴を表示。ユーザー発言・AI応答・画像・Copyボタン付き
- **File Uploader:** txt, csv, json, pdf, 画像, 音声等の複数ファイル添付
- **WEB Search:** チェックボックスとエンジン選択（Perplexity/OpenAI/Google）
- **Include URL Subpages:** URL自動取得時に同一ドメイン内のサブページもクロールするか
- **Personas multiselect:** エージェントに `ORG` が定義されている場合のみ表示。選択中ORG配下から複数ペルソナを選んで並列実行できる（詳細は 3-9）
- **Include Query:** **直前seqが複数ペルソナ実行と判定された場合のみ表示**。ONで前回ペルソナ応答を次ターン入力先頭に埋め込む（RAGクエリ生成には影響なし）
- **Private Mode / Thinking Mode:** チェックボックスで切替。Private ModeはデフォルトTrue（有効）、Thinking ModeはデフォルトFalse（無効）
- **Thinking Targets:** Thinking Mode ON時に表示。`Habit` / `Web Search` / `RAG Query` / `Books` / `Personas`（ORG定義エージェントのみ）から判定対象を選択
- **Max Personas:** Thinking Mode ON かつ Thinking Targets に `Personas` が選ばれているときのみ表示。PersonaSelectorの選定上限（既定 `setting.yaml` の `MAX_PERSONAS=3`）
- **Book multiselect:** エージェントの `BOOK` 設定が **空でない場合のみ表示**。RAGデータを追加選択
- **User Memory expander:** BOOKの直下、`Allowed["User Memory"]: true` のユーザーのみ表示。Active Layers表示 / `persona`・`nowaday`・`history` のチェックボックス（Save押下に関わらず即時反映）+ Save Layer Setting（`users.json` の `Allowed["User Memory Layers"]` に永続保存）。チャットセッション切替でユーザーマスタ値にリセット。メモリ内容の確認・修正は User Memory Explorer の「マイメモリ」タブへ移管（5-5）。詳細は3-10
- **Download:** Markdown/PDFダウンロード（Chats Only / ALL）

> 「Hide irrelevant sidebar controls until they apply」方針: **Max Personas** / **Include Query** / **Book multiselect** はそれぞれ前提条件を満たしたときだけ描画される。前提が崩れた場合（例: ORG無しエージェントへ切替）は保存値も自動的にクリアされる。

## 設定場所

- WebUI本体: `WebDigiMatsuAgent.py`
- system.env: `WEB_TITLE`, `WEB_DEFAULT_AGENT_FILE`, `LOGIN_ENABLE_FLG`, `TIMEZONE`
- setting.yaml: `AGENT_FOLDER`, `SESSION_FOLDER_PREFIX`, `TEMP_FOLDER`

## 設定例

```env
WEB_TITLE=Digital MATSUMOTO
WEB_DEFAULT_AGENT_FILE=agent_standard.json
LOGIN_ENABLE_FLG=Y
TIMEZONE=Asia/Tokyo
```
