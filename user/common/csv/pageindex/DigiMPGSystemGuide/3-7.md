# 3-7. コンテンツコンテキストとWeb検索

## 概要
ユーザーがアップロードしたファイルやWeb検索結果をLLMプロンプトに注入する仕組みである。多様なファイル形式に対応し、Web検索は3種類のエンジンから選択できる。

## 仕組み

### コンテンツコンテキスト
`DigiM_Context.py`の`create_contents_context`関数が、`DigiM_Agent.py`の`set_contents_context`経由で呼ばれる。ファイル形式ごとに処理が分岐する。

- **text**: `read_text_file`でテキストを読み込み、ファイル名付きでコンテキスト化
- **pdf**: `read_pdf_file`でPDFを読み込み、JSON形式でコンテキスト化
- **json**: `read_json_file`で読み込み、JSON文字列としてコンテキスト化
- **image**: `SUPPORT_AGENT.ART_CRITICS`で指定されたエージェントが画像を解釈し、テキスト記述を生成
- **audio**: `mp3_to_text`で音声をテキストに変換
- **office** (`.docx`/`.xlsx`/`.pptx`): 各拡張子に応じたパーサでテキスト抽出

各コンテンツは`<br>---------<br>ファイル名: {name}<br><br>{内容}`の形式で結合され、ユーザークエリに追加される。

### URL自動取得（`DigiM_UrlFetch`）
ユーザー入力に `http(s)://...` リンクが含まれていると、`extract_urls`で抽出 → `fetch_urls_from_text`でページ本文をテキスト化し、上記コンテンツコンテキスト機構に**添付ファイルとして**追加する。

- **サブページクロール**: チャット入力欄上部の「Include URL Subpages」チェックボックス（`st.session_state.url_fetch_subpages`）でON/OFF。ONの場合は同一ドメイン内のリンク先を上限まで再帰取得。
- **安全制御**: プライベートIP・ループバック・リンクローカルへのアクセス禁止、サイズ上限、危険な拡張子（`.exe`/`.dll`等）の取得拒否、ホワイトリスト/ブラックリストドメインによる制御。
- **設定**: `setting.yaml` の `URL_FETCH` ブロック（`TIMEOUT`/`MAX_BYTES`/`MAX_SUBPAGES`/`MAX_DEPTH`/`USER_AGENT`/`ALLOWED_DOMAINS`/`BLOCKED_DOMAINS`/`BLOCKLIST_FILE`/`BLOCKED_EXTENSIONS`）。

### Web検索
`DigiM_Tool.py`に3つのWeb検索エンジンが実装されている。

1. **Perplexity**: `WebSearch_PerplexityAI` - sonarモデルを使用。`setting.yaml`で`PERPLEXITY_URL/MODEL/SYSTEM_PROMPT`等を設定。
2. **OpenAI**: `WebSearch_OpenAI` - `web_search_preview`ツール付きでgpt-4.1-miniを使用。アノテーションからURLを抽出。
3. **Google**: `WebSearch_Google` - Geminiの`GoogleSearch`グラウンディング機能を使用。`grounding_chunks`からURLを抽出。

`WEB_SEARCH_ENGINES`辞書でディスパッチされ、`WebSearch`関数が`engine`引数で切り替える。検索結果は`[参考]関連するWEBの検索結果:`としてクエリに追加される。

### Private Mode
`execution["PRIVATE_MODE"]`で制御される。有効時はRAGデータにprivateフラグを設定し、検索対象を制限する。WebUIではデフォルトが**True**（有効）に設定されている（`st.session_state.private_mode = True`）。

## 設定場所
- Web検索エンジン: `setting.yaml`の各`*_MODEL`, `*_SYSTEM_PROMPT`等
- デフォルトエンジン: `setting.yaml`の`WEB_SEARCH_DEFAULT`
- 実行時: `execution["WEB_SEARCH"]`と`execution["WEB_SEARCH_ENGINE"]`
- Private Mode: `execution["PRIVATE_MODE"]`（WebUIデフォルト: True）

## 設定例
```yaml
WEB_SEARCH_DEFAULT: "Perplexity"
PERPLEXITY_MODEL: "sonar"
OPENAI_SEARCH_MODEL: "gpt-4.1-mini"
GOOGLE_SEARCH_MODEL: "gemini-2.5-flash-preview-05-20"
```
