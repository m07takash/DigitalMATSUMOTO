# 口調テンプレート

## 概要
プロンプトテンプレートマスター（`sample_prompt_templates.json`）のSPEAKING_STYLEセクションに定義された口調テンプレートを、エージェントのシステムプロンプトに注入する機能。エージェントの応答の言葉遣い・トーンを制御する。

## 仕組み
`DigiM_Agent.py`の`set_system_prompt()`内で、`PERSONALITY.SPEAKING_STYLE`が設定されている場合に以下の処理が行われる。

1. プロンプトテンプレートマスターJSON（環境変数`PROMPT_TEMPLATE_MST_FILE`で指定）を読み込む。
2. `SPEAKING_STYLE`セクションから、指定されたキーに対応する説明文を取得する。
3. システムプロンプトに「口調:{説明文}」として追記する。

### 利用可能なスタイル一覧

| キー | 説明 |
|------|------|
| `Polite` | 「です、ます」等の丁寧な言葉遣い |
| `Honorific` | 相手に対して慎重に尊敬語で返す |
| `Strong` | 「だ、である」等のように言い切る強い口調 |
| `Light` | 友達同士の日常会話の様な軽い口調 |
| `Samurai` | 江戸時代の侍が使っているような口調 |
| `Gal` | 渋谷や六本木で夜中までクラブで遊ぶのが好きな若いギャルの口調 |
| `Otaku` | アニメやアイドルが好きでブツブツと独り言のように話すオタクの口調 |

スタイルを追加する場合は、マスターJSONの`SPEAKING_STYLE`セクションにキーと説明文を追加するだけでよい。

## 設定場所
- 口調マスター定義: `user/common/mst/sample_prompt_templates.json` の `SPEAKING_STYLE`
- エージェント側の指定: エージェントJSON の `PERSONALITY.SPEAKING_STYLE`
- マスターファイルパス: 環境変数`PROMPT_TEMPLATE_MST_FILE`（`MST_FOLDER`内）

## 設定例
```json
// エージェントJSON
"PERSONALITY": {
  "SPEAKING_STYLE": "Polite"
}

// マスターJSON（SPEAKING_STYLEセクション）
"SPEAKING_STYLE": {
  "Polite": "「です、ます」等の丁寧な言葉遣い",
  "MyCustomStyle": "関西弁で親しみやすく話す口調"
}
```
