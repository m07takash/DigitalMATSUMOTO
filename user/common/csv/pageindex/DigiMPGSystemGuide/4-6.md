# 4-6. Knowledge Explorer（知識活用分析）

## 概要
RAGデータの中身を **Overall / Trend / Topic / Ask Agent / Generate Report** の5セクションで可視化・分析する WebUI 画面。サイドバーのラジオ（Chat / **Knowledge Explorer** / User Memory Explorer / Scheduler）から切り替える。Collection（VectorDB）と PageIndex の2モードに対応。バックエンドは `DigiM_VAnalytics.py` と `WebDigiMatsuAgent.py` の `_knowledge_explorer()`。

## 仕組み

### データソース選択
- **Collection（VectorDB）**: 選択中エージェントの `KNOWLEDGE`/`BOOK` に紐付くRAGコレクションを一覧。すべての分析は **Total + RAG NAMEごと** に固定で出力する。
- **PageIndex**: ページの階層をツリー構造で可視化。ページ感度分析（LLMによるページ選択シミュレーション）を実施。

重い図（散布図・棒グラフ・ワードクラウド等）は実行時に一度だけ画像化し、再描画では再計算しない。

### 1. Overall
カラムフィルタ・ワイルドカード（`*`）テキスト検索・日付範囲・Private除外で母集団を絞り、**Search & Plot** で散布図とCSVを一括生成。次元削減は **PCA / t-SNE**、Color By・Marker By・Dot Size（Uniform / Newer=Larger / Highlight Period）を選択可能。続けて **Clustering**（K-Means / DBSCAN(eps自動推定) / 階層的）を Total で実施し、各 RAG NAME には Total のクラスタ割当・色をそのまま適用（RAG単位の再クラスタリングはしない）。**Clustering解説** は `agent_23DataAnalyst.json` + 「Cluster Analyst KE」テンプレで生成し、複数バージョンをドロップダウンで切替できる。

### 2. Trend
`create_date` がある場合のみ表示。期間単位（月/四半期/年）と Category Column を選択し、Total / 各RAG NAME に「積み上げ棒 → 期間別TF-IDFキーワード → ワードクラウド（Period降順）」を出力。**Trend解説** はナラティブ文章＋根拠付きの今後推定で、テンプレ「Trend Analyst」を使用。任意の `Focus Period From/To` で対象期間の特徴を中心に解説させられる。

### 3. Topic
クエリテキストに反応する知識チャンクを分析（クラスタリングなし）。Top N と Date Bonus（Bonus Period From/To）を指定し、横軸 Period の二軸チャート（件数バー + 類似スコア合計/平均/最大の折れ線）と散布図を Total / 各RAG NAME で出力。**Topic解説** はテンプレ「Sensitivity Analyst」+ `agent_23DataAnalyst.json`。

### 4. Ask Agent
チャットと同じパイプライン（`DigiMatsuExecute_Practice`）でエージェントに質問。Overall/Trend/Topic の表示中の解説が文脈として渡される。Web Search / Private Mode / Thinking Mode / Books / 会話履歴保持 / Detail Information / Analytics Results の各オプションを利用できる。Analytics Results では同質問の知識活用分析（`analytics_knowledge`）とエージェント比較（別エージェント・別エンジンで実行し、応答のコサイン距離 + テキスト比較を表示）が可能。

### 5. Generate Report / セッション管理
**Generate Report** で `user/common/analytics/knowledge_explorer/analyticsYYYYMMDD_HHMMSS/` に Markdown + 画像 + `state.pkl` を保存。各セクションは「ドロップダウンで表示中のバージョン」がそのまま出力される。サイドバーから保存済みセッションを選んで pickle 復元できる。

### 知識活用分析（`DigiM_VAnalytics.analytics_knowledge`）
- `knowledge_utility = similarity_Q - similarity_A` を質問ベクトル⇄知識／回答ベクトル⇄知識の差分で算出。
- 正規化モード: `Default` / `Norm(All)` / `Norm(Group)`。次元削減: PCA / t-SNE（perplexity自動調整）。
- RAGごとに min/mean/median/max/variance を集計し、横棒・散布図・カテゴリ別散布図を出力。

## 設定場所
- WebUI: `WebDigiMatsuAgent.py` の `_knowledge_explorer()`
- 分析ロジック: `DigiM_VAnalytics.py`（`analytics_knowledge` / `create_similarity_plot_file` 等）
- 解説エージェント: `agent_23DataAnalyst.json` + プロンプトテンプレ「Cluster Analyst KE」「Trend Analyst」「Sensitivity Analyst」
- カテゴリマップ: `user/common/mst/category_map.json`
- 権限: `users.json` の `Allowed["Knowledge Explorer"]`（Adminは常時可）

## 設定例
```json
"RealMatsumoto": {
  "Allowed": {
    "Knowledge Explorer": true,
    "Analytics Knowledge": true,
    "Analytics Compare": true
  }
}
```
