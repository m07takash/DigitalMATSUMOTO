# Big5性格モデル

## 概要
ビッグファイブ（Big Five）性格特性モデルをエージェントのシステムプロンプトに注入し、LLMの応答傾向を性格的に制御する機能。5つの特性に0〜1のスケールで値を設定する。

## 仕組み
`DigiM_Agent.py`の`set_system_prompt()`内で、`PERSONALITY.BIG5`が存在し空でない場合に以下のフォーマットでシステムプロンプトに追記される。

```
ビックファイブ:[Openness:70.00%, Conscientiousness:70.00%, Extraversion:70.00%, Agreeableness:70.00%, Neuroticism:20.00%]
```

各値は0〜1のスケールで定義し、表示時に100倍してパーセント表記される（小数2桁）。

### 5つの特性

- **Openness（開放性）**: 新しい経験やアイデアへの受容度。高いほど創造的・好奇心旺盛な応答になる。
- **Conscientiousness（誠実性）**: 計画性・責任感の度合い。高いほど体系的・正確な応答になる。
- **Extraversion（外向性）**: 社交性・活発さの度合い。高いほど積極的・表現豊かな応答になる。
- **Agreeableness（協調性）**: 他者への配慮・協力度合い。高いほど共感的・穏やかな応答になる。
- **Neuroticism（神経症傾向）**: 感情の不安定さの度合い。高いほど慎重・懸念を示す応答になる。

### 値の目安
- 0.0〜0.3: 低い（その特性が弱い）
- 0.4〜0.6: 中程度
- 0.7〜1.0: 高い（その特性が強い）

## 設定場所
- エージェントJSON: `PERSONALITY.BIG5`
- 処理関数: `DigiM_Agent.py` の `set_system_prompt()`

## 設定例
```json
"BIG5": {
  "Openness": 0.7,
  "Conscientiousness": 0.7,
  "Extraversion": 0.7,
  "Agreeableness": 0.7,
  "Neuroticism": 0.2
}
```

上記の設定は、創造的で誠実、社交的で協調性が高く、情緒安定したキャラクターを表現する。Neuroticismを低くすることで落ち着いた印象を与える。
